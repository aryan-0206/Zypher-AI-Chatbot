## 📋 Prerequisites You'll Need

1. **Python 3.8+** 
   - Download: https://www.python.org/downloads/
   - ⚠️ Important: Check "Add Python to PATH" on Windows

2. **Node.js 16+**
   - Download: https://nodejs.org/ (Get the LTS version)

## ⚡ Ultra Quick Start

```bash
# Extract ZIP to your Desktop

# Terminal 1 - Backend
cd Desktop/ai-chatbot/backend
python -m venv venv
venv\Scripts\activate              # Windows
source venv/bin/activate           # Mac/Linux
pip install -r requirements.txt
python app.py

# Terminal 2 - Frontend
cd Desktop/ai-chatbot/frontend
npm install
npm run dev

# Open browser: http://localhost:3000
```

## ❓ Need Help?

### Common Issues:
- **"Python not found"** → Make sure Python is installed and in PATH
- **"npm not found"** → Make sure Node.js is installed
- **"Port in use"** → Close other terminals or restart computer
- **Errors in terminal** → Read INSTALLATION_GUIDE.md troubleshooting section

### Getting Support:
1. Check INSTALLATION_GUIDE.md for detailed help
2. Read error messages carefully
3. Ensure Python and Node.js are correctly installed
4. Try clean installation (delete venv and node_modules folders)
